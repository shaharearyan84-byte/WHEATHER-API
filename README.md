# WHEATHER-API
# 🌤 Weather Forecast App

A modern and responsive Weather Forecast Web Application built using **Python Flask**, **HTML**, **CSS**, and **JavaScript**. This application fetches real-time weather data using the OpenWeather API and displays temperature, humidity, wind speed, and weather conditions in an attractive glassmorphism user interface.

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡 Real-time temperature display
* 💧 Humidity information
* 🌬 Wind speed details
* ☁ Weather condition description
* 🎨 Animated colorful gradient background
* 🕒 Live digital clock
* ✨ Glassmorphism UI design
* 📱 Responsive layout
* ⚡ Fast and lightweight Flask backend

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/weather-api.git
cd weather-api
```

### Install Required Packages

```bash
pip install flask requests
```

---

## 🔑 OpenWeather API Setup

1. Create an account on OpenWeather.
2. Generate an API key.
3. Open `app.py`.
4. Replace:

```python
API_KEY = "YOUR_OPENWEATHER_API_KEY"
```

with your actual API key.

---

## ▶ Run the Application

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
